## 🔥 关键业务场景测试

### 场景1：完整营销活动流程
1. 登录系统
2. 创建/导入联系人
3. 创建标签并为联系人分类
4. 创建邮件模板（富文本+变量）
5. 创建发送任务关联模板和联系人
6. 执行发送并监控状态
7. 查看发送统计和结果

### 场景2：联系人全生命周期管理
1. 导入联系人列表
2. 批量添加标签
3. 发送欢迎邮件
4. 跟踪邮件状态
5. 处理退信和投诉
6. 更新联系人状态
7. 导出活跃联系人

### 场景3：模板和任务协作流程
1. 创建基础邮件模板
2. 复制模板创建变体
3. 为不同标签组创建对应任务
4. 设置不同发送时间
5. 监控各任务执行情况
6. 对比不同模板效果
